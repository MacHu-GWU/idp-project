Welcome to ``idp`` Documentation
==============================================================================
A python library for Intelligent Document Processing.


.. _install:

Install
------------------------------------------------------------------------------

``idp`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install idp

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade idp
